Repo-internal drift; shipped template is clean.
- **#1 server-source-disclosure (prior CRIT)** is now MITIGATED in the current tree at the framework layer (`staticRoutes.ts` `SERVE_DENYLIST_REGEX` 404s `/server.js` + `*.map` before any `serveFile`), with a consumer defense-in-depth denylist too. Caveat: both fixes are UNCOMMITTED working-tree changes; the server bundle is still emitted into the served `dist/`. Treated as FIXED-in-tree, with the allow-by-default model (#2) left OPEN.
