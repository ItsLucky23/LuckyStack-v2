# @luckystack/router

> Tier-B standalone HTTP + WebSocket load-balancer for multi-instance [LuckyStack](https://github.com/ItsLucky23/LuckyStack-v2) deployments. Redis-backed health, boot-UUID handshake, fallback proxy. Kept in the monorepo for shared tooling but not published — projects that need horizontal scaling fork or vendor it directly.
